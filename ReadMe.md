This is an on going project

A Discord with minimal effort and time!

If you encounter any **bugs or issues** throughout the process of trying to get your bot running, then please **create an issue on the github**, and we'll check into it, as soon as we can.

Step.1 install Node.js.
Step.2 install notepad++.
Step.3 Run "install.bat" file.
Step.4 Add your discord bot token in the "config.json" file.
Step.5 Customise your bot using the "index.js" file.
Step.6 Runt the "run.bat" file.
